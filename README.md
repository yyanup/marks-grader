
# Student Marks Analysis System

This is a Python and NumPy based mini project that analyzes student marks and calculates total, average, highest marks, lowest marks, and grades.

## Features
- Calculates total marks
- Finds average marks
- Displays highest and lowest marks
- Assigns grades automatically

## Technologies Used
- Python
- NumPy

## How to Run

Install NumPy:

```bash
pip install numpy

Run the program:

python filename.py

Average Marks: 83.4
Grade: B
